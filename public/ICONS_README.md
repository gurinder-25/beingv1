# App Icons

For the PWA to work properly, you need PNG icons.

## Quick Setup:

1. **Convert the SVG icons to PNG:**
   - Go to https://cloudconvert.com/svg-to-png
   - Upload `icon-192.svg` and convert to PNG (192x192)
   - Upload `icon-512.svg` and convert to PNG (512x512)
   - Save as `icon-192.png` and `icon-512.png` in the `public` folder

2. **Or use any image editor:**
   - Create 192x192px and 512x512px PNG images
   - Black background with white "B" or your logo
   - Save in the `public` folder

3. **Or use online tools:**
   - https://www.pwabuilder.com/imageGenerator
   - Upload any square image and it will generate all sizes

The SVG files are just placeholders - replace with proper PNG icons for production!
